# Contract App

This is a Hardhat project for the smart contracts of the application.

## Setup

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Compile contracts:

   ```bash
   npx hardhat compile
   ```

3. Run tests:

   ```bash
   npx hardhat test
   ```

4. Deploy:
   ```bash
   npx hardhat run scripts/deploy.ts
   ```
